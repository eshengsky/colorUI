# 定位

使用这些速记工具类可以快速配置元素的位置。

```html
<div class="static">...</div>
<div class="relative">...</div>
<div class="absolute">...</div>
<div class="fixed">...</div>
<div class="sticky">...</div>
```

## 顶部固定

```html
<div class="fixed-top">...</div>
```

## 底部固定

```html
<div class="fixed-bottom">...</div>
```

## 顶部黏性固定

```html
<div class="sticky-top">...</div>
```
